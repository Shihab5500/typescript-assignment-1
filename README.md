# TypeScript Problem Solving Assignment

## Description
This repository contains the solutions for Assignment 1. It includes 7 TypeScript problem-solving tasks and 2 technical blog posts about TypeScript features.

## File Structure
- `solutions.ts`: Contains all the TypeScript code for the 7 problems.
- `blog-1.md`: Blog post explaining `any` vs `unknown` types.
- `blog-2.md`: Blog post explaining `Pick` and `Omit` utility types.

## How to Run the Code
If you want to test the code on your local machine, follow these steps:
1. Open your terminal in the project folder.
2. Compile the TypeScript file using the command: `tsc solutions.ts`
3. Run the compiled JavaScript file using the command: `node solutions.js`
